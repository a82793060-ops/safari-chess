# شطرنج السفاري 🦁 — Safari Chess

لعبة شطرنج كرتونية ضد بوتات على هيئة حيوانات، من الكتكوت (400) حتى التنين (2000)، أو ضد صديق عبر رابط دعوة.

## 🎮 العب الآن

**https://a82793060-ops.github.io/safari-chess/**

## التشغيل محليا

الطريقة الأسهل: افتح ملف `index.html` مباشرة في المتصفح.

إن لم يعمل محرك البوت بهذه الطريقة، شغّل خادما محليا من داخل المجلد:

```bash
python3 -m http.server 4173
```

ثم افتح `http://localhost:4173` في المتصفح.

ملاحظة: اللعبة تحتاج اتصال إنترنت عند التشغيل لتحميل محرك Stockfish والخطوط.

## المزايا

- 9 بوتات بمستويات من 400 إلى 2000، لكل بوت حيوان بشخصية وجُمل حوارية خاصة
- اللعب مع صديق عبر رابط دعوة (اتصال مباشر بين المتصفحين عبر WebRTC)
- اختيار اللون: أبيض / أسود / عشوائي
- محرك Stockfish حقيقي مع أخطاء متعمدة للمستويات الضعيفة
- واجهة ثنائية اللغة (عربي/إنجليزي) مع دعم كامل للاتجاهين
- سحب وإفلات + نقر للتحريك، ترقية البيادق، تلميح، تراجع
- أصوات النقلات الكلاسيكية، وكونفيتي عند الفوز
